How the interval schedule works

  1. Set your target impact/arrival using exact UTC time or a countdown timer.
  2. Pick the delay between rally starts, such as 10, 15, or 30 seconds.
  3. The planner spaces players by that interval and calculates each player’s expected arrival using their march time.
  4. Use buffer to make the target arrival earlier by a few seconds. This changes the schedule and the copied plan.
  5. Sort by longest march first for most rally timing plans, or use list order if you want to control the exact player sequence yourself.
